How to restore

If the database is ever lost or you move to a new PC, restore a backup file with this command (run it in PowerShell from the project folder): 

Get-Content backup.sql | docker exec -i hardware-erp-db psql -U erp -d hardware_erp

Replace backup.sql with the file you want to restore. Restoring writes the data back into the database — do it on a fresh/empty database to avoid duplicate records.
